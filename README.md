# php-docker

## Installation

Add the files in [HOST_PATH](HOST_PATH) to your PATH

## Commands

### php-docker-composer

```shell
php-docker-composer ...
```

### php-docker-php

```shell
php-docker-php ...
```

### php-docker-sh

```shell
php-docker-sh ...
```

### php-docker-sh-as-root

```shell
php-docker-sh-as-root ...
```
